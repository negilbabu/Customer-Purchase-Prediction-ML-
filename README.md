## 📖 Project Overview

This project was developed during my **MSc in Artificial Intelligence at Berlin School of Business and Innovation (BSBI)**. The goal is to build a robust predictive model that determines whether a customer will make a purchase based on historical data. By transforming raw customer data into predictive insights, this project demonstrates how AI can be used to optimize sales funnels and improve marketing ROI.

## 🚀 Machine Learning Pipeline

The project follows a rigorous end-to-end Machine Learning workflow:

- **Exploratory Data Analysis (EDA):** Deep-dive into customer demographics, session data, and historical behavior to identify correlations and patterns.
- **Data Preprocessing:** Implementation of specialized pipelines for handling missing values, encoding categorical variables, and feature scaling.
- **Feature Engineering:** Selecting and transforming variables that have the highest predictive power for purchase conversion.
- **Classification Modeling:** Training and fine-tuning various algorithms (e.g., Logistic Regression, Random Forest, or Gradient Boosting) to achieve the highest accuracy and F1-score.
- **Evaluation:** Utilizing Confusion Matrices and ROC-AUC curves to ensure the model is reliable and avoids bias toward the majority class.

## ✨ Key Insights

- **Behavioral Drivers:** Identification of which features (e.g., time spent on site, previous purchases) most strongly predict a future conversion.
- **Model Reliability:** Benchmarked performance against standard baselines to ensure the solution is ready for real-world business applications.
- **Scalable Design:** The preprocessing pipeline is designed to handle new, incoming customer data for real-time predictions.
